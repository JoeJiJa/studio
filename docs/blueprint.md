# **App Name**: AstroMed

## Core Features:

- Inspirational Banner: Display an inspirational medical-themed banner at the top of the homepage.
- Pareto 80/20 Banner: Display a banner explaining the Pareto 80/20 rule and its relevance to studying.
- Study Streak Tracker: Track consecutive daily visits using localStorage with hydration-safe implementation, displaying a skeleton loader on initial render.
- Suggested Modules Carousel: Display modules like 'Thorax' and 'Oncology' in a horizontally scrollable carousel on the home page.
- Question of the Day: Present a daily medical question to engage users.
- Recently Viewed Carousel: Display a carousel of recently viewed books using localStorage with hydration-safe implementation, displaying a skeleton loader on initial render.
- AI Assistant Chat: Provide a floating chat button that opens a dialog for an AI assistant (Dr. Astro), to guide users through the website based on their year and subject. Dr. Astro uses a tool to provide context specific assistance.

## Style Guidelines:

- Primary color: Deep blue (#2E5266) for a professional and trustworthy feel.
- Background color: Light gray (#F0F4F8), nearly white but soft on the eyes.
- Accent color: Teal (#3E8E7E), a brighter and more saturated color to draw the user's attention.
- Body text: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, ideal for body text
- Headline text: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel; Use 'Space Grotesk' for headlines and 'Inter' for body
- Use a set of consistent and clear icons from ShadCN UI for navigation and key actions.
- Maintain a clean and organized layout with a static header and bottom navigation bar.
- Subtle transitions and animations when navigating between sections and opening accordions to improve UX.
